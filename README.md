# The Descriptives Module

## Overview

<img src='inst/icons/analysis-descriptives.svg' width='149' height='173' align='right'/>

**JASP Descriptives module** is a core module for JASP that provides tools for exploring the data with tables and plots. The Descriptives module offers descriptive statistics, raincloud plots, time series descriptives, flexplot, and a plot builder.


## R Packages

<img src='https://www.r-project.org/logo/Rlogo.svg' width='100' height='78' align='right'/>

The functionality is served by several R packages

- **flexplot** — Flexible visualization of data ([flexplot on CRAN](https://cran.r-project.org/package=flexplot))
- **forecast** — Forecasting functions for time series ([forecast on CRAN](https://cran.r-project.org/package=forecast))
- **ggplot2** — Create elegant data visualisations using the grammar of graphics ([ggplot2 on CRAN](https://cran.r-project.org/package=ggplot2))
- **ggrain** — Raincloud plots using ggplot2 ([ggrain on CRAN](https://cran.r-project.org/package=ggrain))
- **tidyplots** — Tidy data visualization ([tidyplots on CRAN](https://cran.r-project.org/package=tidyplots))
- **patchwork** — Combining multiple ggplots ([patchwork on CRAN](https://cran.r-project.org/package=patchwork))
